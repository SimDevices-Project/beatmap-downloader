关于本软件

SimDevices Beatmap Downloader 是一款 osu! beatmap 下载管理器。当前唯一功能为拦截 osu! 官网的 direct 链接，并自动下载和启动 osu! 进行 beatmap 安装的工具。当前版本使用 sayobot 服务器节点进行下载。

Github 地址： https://github.com/iamapig120/simdevices-beatmap-downloader
